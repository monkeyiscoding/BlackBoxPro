1. Information We Collect

Personal Information

When you use Black Box Pro products or contact us, we may collect:

  • Contact Information: Name, email address, phone number, postal address
  • Business Information: Company name, job title, industry
  • Technical Support: Device serial numbers, software versions, error logs
  • Account Information: Username, password, preferences

Device Information

When you use Black Box Pro hardware and software:

  • Hardware Data: Device model, serial number, firmware version
  • Usage Data: Programming operations, success/failure rates, performance metrics
  • Software Data: Application version, update history, feature usage
  • Network Data: IP address, network configuration (for cloud features)

Website Information

When you visit our website:

  • Browsing Data: Pages visited, time spent, referral sources
  • Device Data: Browser type, operating system, screen resolution
  • Cookies: Session data, preferences, analytics information
